# 答辩代码分层说明

## 1. 核心工作代码（可单独讲解）

这些文件不依赖 Qt，可独立运行：

- `src/Lexer.cpp`, `include/Lexer.h`
- `src/Parser.cpp`, `include/Parser.h`
- `src/SymbolTable.cpp`, `include/SymbolTable.h`
- `src/TypeTable.cpp`, `include/TypeTable.h`
- `src/CompilerFacade.cpp`, `include/CompilerFacade.h`
- `src/VirtualMachine.cpp`, `include/VirtualMachine.h`
- `src/Interpreter.cpp`, `include/Interpreter.h`
- `include/CompileResult.h`, `include/Token.h`, `include/Quadruple.h`

构建产物：

- `MiniPasCore`（静态库）
- `MiniPasCoreCLI`（命令行可执行，纯核心链路）

## 2. 展示/可视化代码（可不讲）

- `src/MainWindow.cpp`, `include/MainWindow.h`
- `src/main.cpp`

构建产物：

- `MiniPasPlus`（Qt 可视化程序）

## 3. 答辩建议运行方式

只演示核心时，直接运行 `MiniPasCoreCLI`：

```powershell
MiniPasCoreCLI.exe your_source.pas
```

或管道输入：

```powershell
Get-Content your_source.pas -Raw | MiniPasCoreCLI.exe
```
