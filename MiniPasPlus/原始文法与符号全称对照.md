## 1.1 完整原始文法（Original Grammar）

说明：

- 所有非终结符都使用 `<...>`。
- 以下文法是“按当前代码实现可识别的完整语法”，不是理想化课本扩展版。

```ebnf
<PROGRAM> → program <IDENTIFIER> <DECLARATION_PART> <COMPOUND_STATEMENT> .

<DECLARATION_PART> → <TYPE_DECLARATION_PART> <FUNCTION_DECLARATION_PART> <VARIABLE_DECLARATION_PART>

<TYPE_DECLARATION_PART> → type <TYPE_DECLARATION_LIST> | ε

<TYPE_DECLARATION_LIST> → <TYPE_DECLARATION> <TYPE_DECLARATION_LIST> | ε

<TYPE_DECLARATION> → <IDENTIFIER> = <TYPE_SPECIFICATION> ;

<TYPE_SPECIFICATION> → <RECORD_TYPE_SPECIFICATION> | <ARRAY_TYPE_SPECIFICATION>

<RECORD_TYPE_SPECIFICATION> → record <FIELD_DECLARATION_LIST> end

<ARRAY_TYPE_SPECIFICATION> → array [ <INTEGER_CONSTANT> .. <INTEGER_CONSTANT> ] of <TYPE_NAME>

<FIELD_DECLARATION_LIST> → <FIELD_DECLARATION> <FIELD_DECLARATION_LIST> | ε

<FIELD_DECLARATION> → <IDENTIFIER> : <BASIC_TYPE> ;

<BASIC_TYPE> → integer | real | char | boolean

<FUNCTION_DECLARATION_PART> → <FUNCTION_DECLARATION> <FUNCTION_DECLARATION_PART> | ε

<FUNCTION_DECLARATION> → function <IDENTIFIER> ( <PARAMETER_LIST_OPT> ) : <TYPE_NAME> ; <VARIABLE_DECLARATION_PART_OPT> <COMPOUND_STATEMENT> ;

<PARAMETER_LIST_OPT> → <PARAMETER_LIST> | ε

<PARAMETER_LIST> → <PARAMETER_DECLARATION> <PARAMETER_DECLARATION_TAIL>

<PARAMETER_DECLARATION_TAIL> → ; <PARAMETER_DECLARATION> <PARAMETER_DECLARATION_TAIL> | ε

<PARAMETER_DECLARATION> → <PASS_MODE_OPT> <IDENTIFIER_LIST> : <TYPE_NAME>

<PASS_MODE_OPT> → var | ε

<VARIABLE_DECLARATION_PART> → var <VARIABLE_DECLARATION_LIST>

<VARIABLE_DECLARATION_PART_OPT> → var <VARIABLE_DECLARATION_LIST> | ε

<VARIABLE_DECLARATION_LIST> → <VARIABLE_DECLARATION> <VARIABLE_DECLARATION_LIST> | ε

<VARIABLE_DECLARATION> → <IDENTIFIER_LIST> : <TYPE_NAME> ;

<IDENTIFIER_LIST> → <IDENTIFIER> <IDENTIFIER_LIST_TAIL>

<IDENTIFIER_LIST_TAIL> → , <IDENTIFIER> <IDENTIFIER_LIST_TAIL> | ε

<TYPE_NAME> → <BASIC_TYPE> | <IDENTIFIER>

<COMPOUND_STATEMENT> → begin <STATEMENT_LIST> end

<STATEMENT_LIST> → <STATEMENT> <STATEMENT_LIST_TAIL> | ε

<STATEMENT_LIST_TAIL> → ; <STATEMENT> <STATEMENT_LIST_TAIL> | ε

<STATEMENT> → <COMPOUND_STATEMENT>
              | <IF_STATEMENT>
              | <WHILE_STATEMENT>
              | <CALL_STATEMENT>
              | <ASSIGNMENT_STATEMENT>

<ASSIGNMENT_STATEMENT> → <LEFT_VALUE> := <EXPRESSION>

<CALL_STATEMENT> → <IDENTIFIER> ( <ARGUMENT_LIST_OPT> )

<IF_STATEMENT> → if <CONDITION> then <STATEMENT> <ELSE_PART_OPT>

<ELSE_PART_OPT> → else <STATEMENT> | ε

<WHILE_STATEMENT> → while <CONDITION> do <STATEMENT>

<CONDITION> → <EXPRESSION> <RELATIONAL_OPERATOR> <EXPRESSION>

<RELATIONAL_OPERATOR> → < | > | = | !=

<LEFT_VALUE> → <IDENTIFIER>
               | <IDENTIFIER> [ <EXPRESSION> ]
               | <IDENTIFIER> . <IDENTIFIER>

<EXPRESSION> → <TERM> <EXPRESSION_TAIL>

<EXPRESSION_TAIL> → + <TERM> <EXPRESSION_TAIL>
                    | - <TERM> <EXPRESSION_TAIL>
                    | ε

<TERM> → <FACTOR> <TERM_TAIL>

<TERM_TAIL> → * <FACTOR> <TERM_TAIL>
              | / <FACTOR> <TERM_TAIL>
              | ε

<FACTOR> → <IDENTIFIER>
           | <IDENTIFIER> [ <EXPRESSION> ]
           | <IDENTIFIER> . <IDENTIFIER>
           | <FUNCTION_CALL_EXPRESSION>
           | <CONSTANT>
           | ( <EXPRESSION> )

<FUNCTION_CALL_EXPRESSION> → <IDENTIFIER> ( <ARGUMENT_LIST_OPT> )

<ARGUMENT_LIST_OPT> → <ARGUMENT_LIST> | ε

<ARGUMENT_LIST> → <EXPRESSION> <ARGUMENT_LIST_TAIL>

<ARGUMENT_LIST_TAIL> → , <EXPRESSION> <ARGUMENT_LIST_TAIL> | ε
```

## 1.2 完整原始文法（中文符号版）

说明：

- 与上一节一一对应，只把非终结符改成中文全称写法。
- 终结符（关键字、运算符、界符）保持源码实际字面量。

```ebnf
<程序> → program <标识符> <声明部分> <复合语句> .

<声明部分> → <类型声明部分> <函数声明部分> <变量声明部分>

<类型声明部分> → type <类型声明列表> | ε

<类型声明列表> → <类型声明> <类型声明列表> | ε

<类型声明> → <标识符> = <类型说明> ;

<类型说明> → <记录类型说明> | <数组类型说明>

<记录类型说明> → record <字段声明列表> end

<数组类型说明> → array [ <整型常量> .. <整型常量> ] of <类型名>

<字段声明列表> → <字段声明> <字段声明列表> | ε

<字段声明> → <标识符> : <基本类型> ;

<基本类型> → integer | real | char | boolean

<函数声明部分> → <函数声明> <函数声明部分> | ε

<函数声明> → function <标识符> ( <形参列表可选> ) : <类型名> ; <变量声明部分可选> <复合语句> ;

<形参列表可选> → <形参列表> | ε

<形参列表> → <形参声明> <形参声明后续>

<形参声明后续> → ; <形参声明> <形参声明后续> | ε

<形参声明> → <传参方式可选> <标识符列表> : <类型名>

<传参方式可选> → var | ε

<变量声明部分> → var <变量声明列表>

<变量声明部分可选> → var <变量声明列表> | ε

<变量声明列表> → <变量声明> <变量声明列表> | ε

<变量声明> → <标识符列表> : <类型名> ;

<标识符列表> → <标识符> <标识符列表后续>

<标识符列表后续> → , <标识符> <标识符列表后续> | ε

<类型名> → <基本类型> | <标识符>

<复合语句> → begin <语句列表> end

<语句列表> → <语句> <语句列表后续> | ε

<语句列表后续> → ; <语句> <语句列表后续> | ε

<语句> → <复合语句>
         | <条件语句>
         | <循环语句>
         | <调用语句>
         | <赋值语句>

<赋值语句> → <左值> := <表达式>

<调用语句> → <标识符> ( <实参列表可选> )

<条件语句> → if <条件表达式> then <语句> <否则部分可选>

<否则部分可选> → else <语句> | ε

<循环语句> → while <条件表达式> do <语句>

<条件表达式> → <表达式> <关系运算符> <表达式>

<关系运算符> → < | > | = | !=

<左值> → <标识符>
        | <标识符> [ <表达式> ]
        | <标识符> . <标识符>

<表达式> → <项> <表达式后续>

<表达式后续> → + <项> <表达式后续>
               | - <项> <表达式后续>
               | ε

<项> → <因子> <项后续>

<项后续> → * <因子> <项后续>
          | / <因子> <项后续>
          | ε

<因子> → <标识符>
         | <标识符> [ <表达式> ]
         | <标识符> . <标识符>
         | <函数调用表达式>
         | <常量>
         | ( <表达式> )

<函数调用表达式> → <标识符> ( <实参列表可选> )

<实参列表可选> → <实参列表> | ε

<实参列表> → <表达式> <实参列表后续>

<实参列表后续> → , <表达式> <实参列表后续> | ε
```

## 2. 词法符号全称（中文）

### 2.1 关键字（Keywords）

- `program`：程序开始关键字
- `type`：类型声明关键字
- `record`：记录类型关键字
- `array`：数组类型关键字
- `of`：数组元素类型关键字
- `function`：函数声明关键字
- `var`：变量声明/引用参数关键字
- `begin`：复合语句开始关键字
- `end`：复合语句结束关键字
- `if`：条件分支关键字
- `then`：条件成立分支关键字
- `else`：条件不成立分支关键字
- `while`：循环关键字
- `do`：循环体开始关键字
- `integer`：整型关键字
- `real`：实型关键字
- `char`：字符型关键字
- `boolean`：布尔型关键字

### 2.2 运算符（Operators）

- `:=`：赋值运算符
- `+`：加法运算符
- `-`：减法运算符
- `*`：乘法运算符
- `/`：除法运算符
- `<`：小于关系运算符
- `>`：大于关系运算符
- `=`：等于关系运算符
- `!=`：不等于关系运算符

### 2.3 界符（Delimiters）

- `(` `)`：圆括号
- `[` `]`：方括号
- `.`：点号
- `..`：区间界符
- `,`：逗号
- `:`：冒号
- `;`：分号

## 3. 中间代码与运行层符号全称（中文）

### 3.1 四元式操作符（Quadruple Operators）

- `program`：程序开始标记
- `end`：程序结束标记
- `:=`：赋值
- `+ - * /`：算术运算
- `< > = !=`：关系运算
- `if`：条件判断入口标记
- `el`：条件分支 else 标记
- `ie`：条件分支结束标记
- `wh`：while 循环入口标记
- `do`：while 条件判断后标记
- `we`：while 循环结束标记
- `param`：实参传递
- `call`：函数调用
- `ret`：函数返回

### 3.2 虚拟机指令操作码（VM Instruction Opcodes）

- `LD`：加载（Load）
- `ST`：存储（Store）
- `MOV`：移动赋值（Move）
- `ADD`：加法（Add）
- `SUB`：减法（Subtract）
- `MUL`：乘法（Multiply）
- `DIV`：除法（Divide）
- `LT`：小于比较（Less Than）
- `GT`：大于比较（Greater Than）
- `EQ`：相等比较（Equal）
- `LE`：小于等于比较（Less or Equal）
- `GE`：大于等于比较（Greater or Equal）
- `NE`：不等比较（Not Equal）
- `AND`：逻辑与（Logical And）
- `OR`：逻辑或（Logical Or）
- `NO`：逻辑非（Logical Not）
- `FJ`：条件为假跳转（False Jump）
- `TJ`：条件为真跳转（True Jump）
- `JMP`：无条件跳转（Unconditional Jump）
- `ARG`：压入实参（Argument Push）
- `CALL`：调用函数（Call）
- `RET`：返回（Return）
- `MOVRET`：读取返回值（Move Return Value）
- `HALT`：停止执行（Halt）
