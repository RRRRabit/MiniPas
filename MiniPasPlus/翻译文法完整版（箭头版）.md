最后更新：2026-05-16

说明：
- 使用 `→` 表示产生式。
- `a1, a2, ...` 为语义动作编号。
- 本文法覆盖当前实现支持的完整主流程（声明、语句、表达式、函数调用、四元式生成）。

## 1. 程序与声明

`<程序> → program <标识符> a1 <声明部分> <复合语句> . a2`

`<声明部分> → <类型声明部分> <函数声明部分> <变量声明部分>`

`<类型声明部分> → type <类型声明列表> | ε`

`<类型声明列表> → <类型声明> <类型声明列表> | ε`

`<类型声明> → <标识符> = <类型说明> a3 ; a4`

`<类型说明> → <记录类型说明> | <数组类型说明>`

`<记录类型说明> → record <字段声明列表> end a5`

`<数组类型说明> → array [ <整型常量> .. <整型常量> ] of <类型名> a6`

`<字段声明列表> → <字段声明> <字段声明列表> | ε`

`<字段声明> → <标识符> : <基本类型> a7 ;`

`<基本类型> → integer | real | char | boolean`

`<函数声明部分> → <函数声明> <函数声明部分> | ε`

`<函数声明> → function <标识符> a8 ( <形参列表可选> ) : <类型名> a9 ; <变量声明部分可选> <复合语句> a10 ;`

`<形参列表可选> → <形参列表> | ε`

`<形参列表> → <形参声明> <形参声明后续>`

`<形参声明后续> → ; <形参声明> <形参声明后续> | ε`

`<形参声明> → <传参方式可选> <标识符列表> : <类型名> a11`

`<传参方式可选> → var | ε`

`<变量声明部分> → var <变量声明列表> a12`

`<变量声明部分可选> → var <变量声明列表> a12 | ε`

`<变量声明列表> → <变量声明> <变量声明列表> | ε`

`<变量声明> → <标识符列表> : <类型名> a13 ;`

`<标识符列表> → <标识符> <标识符列表后续>`

`<标识符列表后续> → , <标识符> <标识符列表后续> | ε`

`<类型名> → <基本类型> | <标识符>`

## 2. 语句

`<复合语句> → begin <语句列表> end`

`<语句列表> → <语句> <语句列表后续> | ε`

`<语句列表后续> → ; <语句> <语句列表后续> | ε`

`<语句> → <复合语句>`

`<语句> → <条件语句>`

`<语句> → <循环语句>`

`<语句> → <调用语句>`

`<语句> → <赋值语句>`

`<赋值语句> → <左值> := <表达式> a14`

`<调用语句> → <标识符> ( <实参列表可选> ) a15`

`<条件语句> → if <条件表达式> a16 then <语句> <否则部分可选> a17`

`<否则部分可选> → else a18 <语句> | ε`

`<循环语句> → while a19 <条件表达式> a20 do <语句> a21`

`<条件表达式> → <表达式> <关系运算符> <表达式> a22`

`<关系运算符> → < | > | = | !=`

`<左值> → <标识符> a23`

`<左值> → <标识符> [ <表达式> ] a24`

`<左值> → <标识符> . <标识符> a25`

## 3. 表达式

`<表达式> → <项> <表达式后续>`

`<表达式后续> → + <项> a26 <表达式后续>`

`<表达式后续> → - <项> a27 <表达式后续>`

`<表达式后续> → ε`

`<项> → <因子> <项后续>`

`<项后续> → * <因子> a28 <项后续>`

`<项后续> → / <因子> a29 <项后续>`

`<项后续> → ε`

`<因子> → <标识符> a30`

`<因子> → <标识符> [ <表达式> ] a31`

`<因子> → <标识符> . <标识符> a32`

`<因子> → <函数调用表达式>`

`<因子> → <常量> a33`

`<因子> → ( <表达式> )`

`<函数调用表达式> → <标识符> ( <实参列表可选> ) a34`

`<实参列表可选> → <实参列表> | ε`

`<实参列表> → <表达式> a35 <实参列表后续>`

`<实参列表后续> → , <表达式> a35 <实参列表后续> | ε`

## 4. 语义动作说明（对应实现行为）

- `a1`：登记程序名，生成 `(program, 程序名, _, _)`
- `a2`：生成 `(end, 程序名, _, _)`
- `a3`：创建类型定义上下文并做重定义检查
- `a4`：类型写入符号表与类型表
- `a5`：记录类型字段表落地
- `a6`：数组类型（上下界、元素类型）落地
- `a7`：字段类型与字段名写入当前记录定义
- `a8`：进入函数声明上下文（参数偏移/局部偏移清零）
- `a9`：登记函数返回类型及入口位置
- `a10`：函数末尾生成 `(ret, 函数名, _, _)`
- `a11`：登记参数（值参/变参、类型、偏移）
- `a12`：变量声明部分完成
- `a13`：每个变量落符号表，计算地址与大小
- `a14`：类型检查并生成 `(:=, 源place, _, 目标place)`
- `a15`：调用语句生成 `param/call`（无返回值落地到 `_`）
- `a16`：生成 `(if, 条件place, _, _)`
- `a17`：生成 `(ie, _, _, _)`
- `a18`：生成 `(el, _, _, _)`
- `a19`：生成 `(wh, _, _, _)`
- `a20`：生成 `(do, 条件place, _, _)`
- `a21`：生成 `(we, _, _, _)`
- `a22`：关系运算生成布尔临时变量四元式
- `a23`：普通标识符左值检查
- `a24`：数组下标类型检查并形成数组元素左值
- `a25`：记录字段存在性检查
- `a26`：加法生成临时变量四元式
- `a27`：减法生成临时变量四元式
- `a28`：乘法生成临时变量四元式
- `a29`：除法生成临时变量四元式
- `a30`：标识符因子求 place/type
- `a31`：数组元素因子求 place/type
- `a32`：记录字段因子求 place/type
- `a33`：常量因子求 place/type
- `a34`：函数调用表达式生成 `call`，并处理返回临时变量
- `a35`：每个实参生成 `(param, 实参place, vf, _)`
