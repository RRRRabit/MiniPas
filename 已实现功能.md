# 已实现的功能（MiniPas）

最后更新：2026-05-19（答辩阅读版）

本文档只记录代码中已经实现并可用的功能，以及当前明确限制。

## 1. 工程形态

- 核心库：`MiniPasCore`（词法、语法语义、四元式优化，并链接独立 `MiniPasPlus/VM` 模块执行程序）。
- 可执行：
- `MiniPasPlus`（Qt 可视化界面）
- `MiniPasCoreCLI`（命令行编译运行）
- 默认构建：`BUILD_UI=OFF`，优先构建命令行核心，便于答辩阅读。
- 主流程：`CompilerFacade::compileAndRun` 串联词法 -> 语法语义 -> 四元式优化 -> VM 执行（失败时回退解释器）。

## 2. 词法分析（Lexer）

- Token 类型：`KEYWORD / IDENTIFIER / CONSTANT / DELIMITER / OPERATOR / END_OF_FILE / ERROR`。
- 关键字：`program var integer real char begin end type record array of function boolean while do if then else`。
- 运算符/界符：`, : ; := * / + - . ( ) = [ ] .. < > !=`。
- 常量：整数、实数、字符常量（如 `'a'`）。
- 注释：`{...}`、`//...`。
- 产物：I 表（标识符）、C 表（常量）与关键字/界符表。
- 错误：非法词法项生成 `ERROR` Token，带行列。

## 3. 语法与语义（Parser）

- 语法骨架：`program id DECL_PART COMPOUND_STMT .`。
- 声明：
- `type`：`record`、`array[low..high] of T`
- `function`：支持形参表、返回类型、局部变量
- `var`：全局与函数局部变量
- 语句：赋值、`if/then/else`、`while/do`、`begin/end`、函数调用。
- 表达式：`+ - * /` 与关系运算 `< > = !=`。
- 语义检查：
- 标识符/类型声明检查
- 赋值兼容检查（`integer -> real` 允许）
- 函数实参与形参数量、类型匹配检查
- 数组下标必须为 `integer`
- record 字段合法性检查
- 产物：符号表、类型表、函数表、参数表、四元式和基本块。
- 活动记录、目标代码跟踪、简单优先分析等展示内容已集中到 `MiniPasPlus/GUI`，VM 执行器已集中到 `MiniPasPlus/VM`，默认 `src/include` 主要保留编译前端和优化主线。

## 4. 中间代码与优化

- 四元式：`program/end`、`:=`、`+ - * /`、关系运算、`if/el/ie`、`wh/do/we`、`param/call/ret`。
- 基本块划分已实现。
- 块内优化已实现：
- 常量折叠
- 常量/复制传播
- 公共子表达式消除
- 代数化简
- 无用临时赋值删除

## 5. VM 执行

- VM 代码已独立放在 `MiniPasPlus/VM/VirtualMachine.h` 和 `MiniPasPlus/VM/VirtualMachine.cpp`，不再放在 `src/include` 主源码目录中。
- 已生成结构化 VM 指令：`MOV/MOVRET/ADD/SUB/MUL/DIV/LT/GT/EQ/NE/LE/GE/FJ/JMP/ARG/CALL/RET/HALT`。
- 已实现函数调用主链路：
- 实参压栈（`ARG`）
- 调用跳转（`CALL`）
- 返回值回填（`MOVRET`）
- 函数入口形参绑定（`ARGi -> 形参名`）
- 已实现循环/分支执行，含最大步数保护与除零报错。
- VM 异常时回退到四元式解释器。
- 目标代码跟踪、简单优先分析、VALL 活动记录等可视化展示代码已从默认核心构建中移出；如需界面展示，由 `MiniPasPlus/GUI` 相关代码单独负责。

## 6. 运行限制（当前）

- `a[i].f`（结构体数组字段访问）仍不支持，前端会报错。
- 数组/record 在 VM 中仍以符号级名字近似表示（如 `a[i]`、`p.x`），非完整地址化内存模型。
- CLI 以“文件路径参数”方式运行时，在当前 Windows 非 ASCII 路径环境下存在打开文件失败问题；交互输入模式可正常编译运行。

覆盖点：
- 基础赋值与算术优先级

- `record` 字段读写
- 数组声明与下标访问

- 注释与字符常量

结果：

- `t09` 得到 `z = 9`
- `t17` 得到 `d = 7`
- `t20` 得到 `ans = 25`
